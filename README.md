# Mission One Website

Modern redesign of missionone.io — built with Next.js 14.

## Deploy to Vercel via GitLab

1. Push this repo to GitLab
2. Go to [vercel.com](https://vercel.com) and sign in with GitLab
3. Click **"Add New Project"** and import this repository
4. Vercel will auto-detect Next.js — click **Deploy**
5. Once live, go to **Settings → Domains** and add `missionone.io`
6. Update your domain DNS to point to Vercel (they'll provide the records)

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
